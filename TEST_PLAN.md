# Composer Test Plan

Generated by `/autoplan` engineering review on 2026-06-21  
Branch: no-branch  
Repo: local/Composer

## Affected Screens

- Startup and recovery chooser: empty project, Band Sketch template, autosave recovery.
- Main DAW workspace: transport, dynamic track list, timeline, clip selection and context panel.
- Piano roll: direct note editing, chord insertion and rhythm preview/apply.
- Waveform editor: record, trim, split, fade, gain and crossfade.
- Device setup: WASAPI/ASIO selection, sample rate, buffer, input meter and monitoring.
- Mixer: volume, pan, mute/solo, effects, send and master limiter.
- Project open/save/relink and WAV/MIDI import/export.
- Installer, upgrade, rollback and uninstall.

## Key Interactions To Verify

- Create an empty project and a Band Sketch project.
- Add, remove, duplicate, rename and reorder instrument/audio tracks.
- Create a MIDI clip; add, drag, resize and delete notes; undo and redo each action.
- Press `Ctrl+K`; preview and insert roots, qualities, extensions, inversions and ranges.
- Generate a rhythm; cancel the preview, apply it, edit generated notes and undo it once.
- Arm an audio track during composition; count in, record, monitor, stop and edit the new clip.
- Save, close and reopen with track order, selection, devices, effects and assets intact.
- Export WAV and MIDI, then verify both in an independent player/DAW.
- Complete the full 8-bar acceptance song using keyboard only.

## Edge Cases

- No audio/MIDI device, busy device, disconnect during take, unsupported rate and buffer.
- Disk full, permission loss, short write, shutdown during finalize and recovery on next launch.
- Empty/zero-length clips, overlapping notes, negative/out-of-range edits and extreme tempo maps.
- Missing, moved, corrupt or traversal-path project assets and sample packs.
- Newer/older/corrupt manifest versions and interrupted migration.
- Rapid repeated record/export/chord commands and stale background results.
- 32 tracks, 16 instruments, large note counts, waveform cache pressure and long recording.
- 1024x700 window, 200% scaling, high contrast, reduced motion and screen reader.

## Critical Paths

1. Clean install -> first launch -> Band Sketch -> four instruments -> 8-bar verse -> save/reopen -> WAV export.
2. MIDI clip -> chord shortcut -> ordinary editable notes -> rhythm generation -> Undo/Redo -> MIDI export.
3. Mid-composition audio track -> record -> simulated disconnect -> preserve/recover take -> edit waveform.
4. Open project with missing assets -> offline tracks -> relink -> playback and save without losing references.
5. Upgrade -> backup/migrate project copy -> validate; on failure reinstall previous signed version and open original.

## Release Gates

- All domain/application branches and typed error mappings covered.
- Golden render and project round-trip fixtures deterministic.
- Audio callback meets budget with zero allocations, locks and disk I/O.
- 30-minute reference soak has zero dropout.
- Clean-VM installer, upgrade, rollback and uninstall pass.
- Target users achieve the 10-minute 8-bar flow at least 4/5 times.
